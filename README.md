# 💰 Expense Tracker

A responsive Expense Tracker built with **HTML, CSS, and JavaScript** that helps users manage their daily expenses. The application supports CRUD operations, category-based filtering, monthly expense summaries, and data persistence using Local Storage.

## 🚀 Features

- ➕ Add new expenses
- ✏️ Edit existing expenses
- 🗑️ Delete expenses
- 💾 Persistent storage using Local Storage
- 📂 Filter expenses by category
- 📊 Expense summary
  - Total Expenses
  - Current Month Expenses
  - Total Number of Expenses
- 📱 Responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
